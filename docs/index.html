VELKA

The Code Sin Judge

Blazing-fast secret scanning written in Rust. Detect leaked credentials, API keys, and tokens across your entire codebase in milliseconds. Zero telemetry. Zero compromise.

$ cargo install velka click to install

Everything you need. Nothing you don't.

One binary. Zero dependencies. Absolute privacy.

🛡️

52+ Detection Rules

AWS, GCP, GitHub, Stripe, Slack, Twilio, SendGrid, private keys, JWTs, and many more. Compiled regex for maximum speed.

🔒

Zero Telemetry

Privacy first. No network calls, no analytics, no tracking. Your secrets stay on your machine. Always.

⚡

High Performance

Memory-mapped I/O, parallel file walking, compiled regex, lock-free channels. Thousands of files in milliseconds.

🧠

ML Classifier

Ensemble scoring with Shannon entropy, character frequency analysis, and structural validation. No ML runtime needed.

💻

LSP Server

Real-time secret detection in your editor. Works with VS Code, Neovim, Helix, Zed, and any LSP-compatible editor.

☸️

K8s & IaC Scanning

Admission webhook that blocks Pods with secrets. Native scanning of Terraform, Kubernetes manifests, and Dockerfiles.

📊

Baseline Management

Save a clean snapshot, then diff future scans against it. Only new secrets are flagged â€” zero noise from known findings.

🔍

AST Scope Analysis

Suppresses false positives from test functions, docstrings, and test files. ~40% fewer false positives out of the box.
